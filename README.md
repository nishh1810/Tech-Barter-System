# 🚀 Tech Barter System – Microservices E-Commerce Platform

## 📌 Overview
Tech Barter System is a scalable e-commerce platform built using Spring Boot microservices and Flutter applications.

It supports multiple roles: User, Seller, and Admin.

---

## 🧠 System Architecture
- Microservices-based backend architecture
- API Gateway for routing and authentication
- Separate Flutter apps for each role

---

## 🧩 Backend Services (Spring Boot)
- API Gateway – request routing and authentication
- Auth Service – login and JWT security
- Seller Service – product and seller management
- User Service – marketplace functionality
- Admin Service – platform control

---

## 📱 Frontend Applications (Flutter)
- User App – browse and purchase products
- Seller App – manage listings
- Admin App – system monitoring

---

## ⚙️ Tech Stack
Backend: Java, Spring Boot, REST APIs  
Frontend: Flutter  
Architecture: Microservices, API Gateway  
Database: MongoDB, MySQL  
Tools: Git, Docker, Postman, CI/CD
